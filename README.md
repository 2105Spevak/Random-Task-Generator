# Random Task Generator

**Измайлова Софья** 

## Описание

Random Task Generator - это консольное приложение для генерации случайных задач, которое помогает разнообразить повседневные дела и найти вдохновение для новых занятий. Приложение использует паттерн Factory для создания задач разных типов и сохраняет историю сгенерированных задач в виде очереди.

### Основные возможности

- ✅ **Генерация случайных задач** из базы данных
- ✅ **Фильтрация** по типу задачи и сложности
- ✅ **Различные типы задач**: работа, спорт, учёба, дом, здоровье
- ✅ **Шкала сложности** (1-3): от лёгких до сложных задач
- ✅ **История генерации** в виде очереди (последние 50 задач)
- ✅ **Добавление своих задач** с валидацией ввода
- ✅ **Статистика** по типам и сложности задач
- ✅ **Сохранение в JSON** (задачи и история сохраняются между сеансами)
- ✅ **Принципы ООП**: инкапсуляция, наследование, полиморфизм
